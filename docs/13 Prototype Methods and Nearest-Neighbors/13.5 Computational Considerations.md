# 13.5 计算上的考虑

| 原文   | [The Elements of Statistical Learning](../book/The Elements of Statistical Learning.pdf) |
| ---- | ---------------------------------------- |
| 翻译   | szcf-weiya                               |
| 时间   | 2017-08-29                               |

最近邻分类规则的一个缺点是通常是它的计算负荷量，无论是寻找最近邻还是存储整个训练集合。当有$N$个观测和$p$个变量时，对每个点最近邻分类需要$Np$次操作来寻找邻域。
