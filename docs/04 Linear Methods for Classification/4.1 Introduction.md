# 4.1 导言

在这章我们重新回到分类问题并且注重在分类的线性方法。因为我们的预测变量$G(x)$是在离散集$\cal G$中取值，我们总是可以把输入空间分成一些区域的集合，这些区域根据分类来打上标签。我们在第2章看到这些区域的边界依赖于预测函数，边界会变得粗糙或者光滑。对于这些过程的一个很重要的类，这些判别边界（decision boundaries）都是线性的；这也是我们说的分类的线性方法。
